# MouseWheel

通过reg文件写入，修改注册表，实现鼠标滚轮方向修改，使得win与mac系统滚动方向一致。

编译环境：vs2005 + c#

程序下载： https://sanhuo.info/


    MouseWheel_V1.0.3

    MouseWheel_V1.0.2
    　　修复找不到备份路径
    MouseWheel_V1.0.1
    　　显示多个鼠标设备表情符
 
    W_W 表示当前鼠标为 Windows 风格， O_O 则为 Mac OSX 风格
    第一次运行会备份系统设置，若异常可双击恢复
